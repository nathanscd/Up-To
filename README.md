# Up To - Conexão por Atividades Locais

Up To é um aplicativo que conecta pessoas com base em interesses e atividades próximas, permitindo criar e participar de micro-eventos instantâneos na sua região. Ideal para quem quer fazer novas amizades, encontrar parceiros para hobbies, esportes, estudos ou até mesmo engajar em voluntariado.

## Funcionalidades

* Descubra pessoas e atividades próximas usando geolocalização.
* Crie ou participe de micro-eventos instantâneos.
* Filtros por interesse, disponibilidade e localização.
* Integração com estabelecimentos parceiros para eventos patrocinados.
* Perfis de usuários com preferências e histórico de atividades.
* Sistema de notificações para eventos relevantes em tempo real.

## Diferenciais

* Eventos instantâneos baseados em interesses e horários.
* Conexão com pessoas que têm hobbies ou objetivos semelhantes.
* Experiência social segura e dinâmica.

## Tecnologias

* Backend: Python (Flask)
* Banco de dados: PostgreSQL
* Frontend: React Native (iOS e Android)
* Geolocalização: APIs nativas do dispositivo ou Google Maps
* Notificações em tempo real: Firebase Cloud Messaging

## Instalação

1. Clone o repositório:

```
git clone https://github.com/seuusuario/upto.git
```

2. Instale dependências:

```
pip install -r requirements.txt   # Para backend em Python
npm install                        # Para frontend em React Native
```

3. Configure variáveis de ambiente (API keys, banco de dados, etc.)

4. Execute a aplicação:

```
python app.py   # Backend
npx react-native run-android/run-ios  # Frontend
```

## Uso

* Crie uma conta e configure suas preferências.
* Explore eventos próximos ou crie o seu próprio.
* Interaja com outros usuários e participe de atividades de interesse.
